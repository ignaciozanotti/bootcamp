Read a little bit before dropping the console:

  • A couple of movies are already created. You can play(), stop(), getTitle(), setTitle(newtitle), share(name) and like() terminator and robocop.
    ie: type "terminator.play();" in the console

  • robocop is a DownloadableMovie, this means you can download it by typing:
    "robocop.download();" in the console

  • you can create a new movie as follows:
    var thegodfather = new Movie ('the god father', 1972);

  • don't forget to add an observer to the movie, you can create your own or make use of the one provided (theObserver):
    thegodfather.AddObserver(theObserver);
